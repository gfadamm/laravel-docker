# Laravel Docker

## API Documentation

Postman Collection tersedia di:

```text
docs/Laravel Docker.postman_collection.json
```

Cara menggunakannya:

1. Buka Postman.
2. Klik **Import**.
3. Pilih file `docs/Laravel Docker.postman_collection.json`.
4. Atur `base_url` sesuai environment yang digunakan (misalnya `http://laravel.test`).
